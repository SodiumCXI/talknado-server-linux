## Version Compatibility

| Server Version | Supported Client Versions      |
|----------------|--------------------------------|
| v1.1.2         | v1.2.4, v1.2.5, v1.3.0, v1.3.1 |
| v1.1.3         | v1.3.2, v1.3.3                 |

## Installation

### Install the latest version

```bash
curl -fsSL https://raw.githubusercontent.com/SodiumCXI/talknado-server-linux/main/install.sh | bash
```

### Install a specific version

Replace **vX.X.X** with the desired server version:

```bash
curl -fsSL https://raw.githubusercontent.com/SodiumCXI/talknado-server-linux/main/install.sh | bash -s vX.X.X
```
